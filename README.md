# Project1

A simple website hosted on GitHub Pages.

## 🌐 Live Website

Visit the live website at: https://youn-17.github.io/Project1/

## 📁 Project Structure

- `index.html` - Main webpage
- `.github/workflows/pages.yml` - GitHub Actions workflow for automatic deployment

## 🚀 GitHub Pages Deployment

This site is automatically deployed to GitHub Pages whenever changes are pushed to the main branch. The deployment is handled by GitHub Actions.

## 🛠️ Local Development

To view the site locally, simply open `index.html` in your web browser.

## 📝 Customization

You can customize the website by editing:
- `index.html` - Main page content and structure
- Add CSS files for styling
- Add JavaScript files for interactivity
- Create additional HTML pages as needed
